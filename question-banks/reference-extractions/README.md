# Reference extractions

These JSON files contain extracted source text for later curation. They are not listed in the active `question-banks/index.json` manifest because source-reference entries may not have a static answer that QuizHub can grade.

Each bank preserves source attribution and license metadata. Review the license and source path before adapting or redistributing an entry. New and maintained banks use `zh-Hans` and `zh-Hant`; redundant `zh` fields have been removed.

The computer-science reference bank covers CPU cache (including L1/L2), digital systems, operating systems, TCP/IP layers, routing, DNS, congestion, security/privacy, and distributed-system reliability. It contains 9 source-reference entries from OpenStax Introduction to Computer Science, MIT OpenCourseWare 6.004/6.033, and Engineering LibreTexts Networks. These entries are study prompts and do not count toward scored quizzes until a verified answer is added.

The Malaysia examination-materials reference bank records ACCA TX-MYS sample questions with an official answer/marking-scheme link, LHDN MTD Testing Questions 2026, and the MIA past-years Q&A page. It intentionally contains topic summaries and source links rather than copied exam text; ACCA and MIA materials have reuse or access restrictions.
